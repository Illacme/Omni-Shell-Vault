# 🛡️ Omni-Shell-Vault

A collection of industrial-grade shell scripts for AI infrastructure, diagnostics, and high-performance automation.

---

## 🚀 Featured Tool: Gemini API Sentinel (v9.1 Global Edition)

The flagship diagnostic tool in this vault. It provides full-matrix auditing for the Gemini API ecosystem.

### Key Features
- **Smart Matrix Scanning**: Automatically discovers all authorized models for your API key.
- **Autonomous Error Handling**: Intelligent 429 (Rate Limit) backoff with `retryDelay` parsing.
- **Cross-Platform Efficiency**: Runs flawlessly on macOS, Linux, and specialized cloud environments.
- **High-Integrity Reporting**: Generates a mission-control style ASCII summary table for quick audits.

### Quick Start (快速开始)
1. **Clone the vault / 克隆仓库**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Omni-Shell-Vault.git
   cd Omni-Shell-Vault
   ```
2. **Assign execution permissions / 授权**:
   ```bash
   chmod +x check_gemini.sh
   ```
3. **Run a full audit / 全量审计**:
   ```bash
   ./check_gemini.sh -k YOUR_API_KEY -a
   ```
